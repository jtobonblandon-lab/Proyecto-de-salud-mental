# Proyecto: Red de Apoyo contra la Violencia de Género

Este proyecto incluye páginas informativas, un formulario de contacto y un test interactivo (Violentómetro). 

## Estructura
- index.html
- violenciadegenero.html
- violencia-sexual.html
- violencia-economica.html
- violencia-fisica.html
- violencia-psicologica.html
- rutasdeapoyo.html
- red-de-apoyo-a-genero.html
- buscasorientacion.html
- violentometro.html
- styles.css
- styles2.css
- juego.js
- imagenes/violentometro.png

## Cómo usar
1. Abre `index.html` en tu navegador.
2. Navega por el menú superior.
3. En `Violentómetro`, inicia el test y revisa el resultado.

## Paleta
Se mantiene la paleta lila original, con mejoras de accesibilidad y responsividad.
